\begin{frame}
    \frametitle{Discussion and Limitations}
    \framesubtitle{Discussion}
    \begin{itemize}
        \item one-step pre-trained models can serve as a backbone model for many image synthesis tasks
        \item Adapting the models can be achieved through GAN objectives without multi-step diffusion training
        \item model training requires a small number of additional trainable parameters
    \end{itemize}
\end{frame}
\note{
    \begin{itemize}
        \item one-step pre-trained Modelle können als Backbone-Modelle für viele Bildsynthesen-Aufgaben dienen.
        \item Die Anpassung der Modelle kann durch GAN-Objektive ohne mehrstufiges Diffusionstraining erreicht werden.
        \item Das Training des Modells erfordert eine geringe Anzahl zusätzlicher trainierbarer Parameter.
    \end{itemize}

}
    
% ---------- Limitations ----------
\begin{frame}
    \frametitle{Discussion and Limitations}
    \framesubtitle{Limitations}
    \begin{itemize}
        \item cannot specify strength of guidance as SD-Turbo does not use classifier-free guidance
        \item does not support negative prompt
        \item training is memory intensive
    \end{itemize}
    
\end{frame}
\note{
    \begin{itemize}
        \item Die Stärke der Anleitung kann nicht spezifiziert werden, da SD-Turbo keine classifier-free guidance verwendet.
        \item Negative Prompts werden nicht unterstützt, wodurch Bild Artefakte reduziert werden könnten.
        \item Das Training ist durch den cycle-consistency loss speicherintensiv.
    \end{itemize}
}
